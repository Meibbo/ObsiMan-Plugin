Milestone v1.0.0 — Scope completo

Items confirmados para el release. Continúa respondiendo A / B / C abajo para el brainstorm de context menus.

Navegación y estructura

  • Hub bar (actual "tab bar") → muestra Pages. Badge counters se muestran aquí también.
  • Pill bar → muestra Tabs dentro de la page activa. Sin labels; solo hover.
  • Filters bar (renombrado desde "filters header") — se trata como un bar más en el sistema.
  • Content tab (renombrado desde "Files Ops") → pasa a Filters page. Orden default: Props · Tags · Files · Content

Interacción con nodos (tree / grid)

  • Click simple → seleccionar nodo
  • Doble click / Ctrl+click → abrir nota (Files) o enviar a Find & Replace (Props / Tags)
  • Keyboard navigation queda para v1.1+

Tree view — CSS fixes

  • Texto y caret usan variable más oscura de Obsidian (--text-muted o similar), no el blanco de notas
  • Eliminar background del counter badge en los nodos (quitar el pill/chip visual)
  • Separación ligeramente mayor entre nodos (user ajusta manualmente después)
  • Nodos sin ícono (ej. valores de props): el icon slot no ocupa espacio horizontal
  • Líneas guía de indentación clásicas en árbol (como VS Code / file explorer)

Popups y FABs

  • View modal y Sort modal de Filters bar → requieren su propio brainstorm, siguiendo fielmente el frame
  • FAB popups (Queue list + Active filters) → mantienen continuidad visual actual
  • Diseño actual de Active Filters → pasará a "helper island / notification system" en v1.1+
