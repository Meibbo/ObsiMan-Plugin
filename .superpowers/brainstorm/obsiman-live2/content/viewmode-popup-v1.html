View Mode Popup — Layout

El popup reemplaza el header cuando se presiona el botón izquierdo del Filters header.
Entra desde la DERECHA (el header sale a la izquierda). Propón el cierre con ‹ a la izquierda.

Opción A
Single row — pills + hub compacto
Todo en una fila. Pills de view mode centradas, squircle hub al extremo derecho que despliega una fila extra de toggles al presionarlo.

  • Muy compacto — row 2 solo aparece si se necesita

Opción B
Two rows siempre — pills arriba, hub items abajo
Dos filas fijas (como el Sort popup). Fila 1: close + pills. Fila 2: hub siempre visible con los toggles del tab activo. Sin squircle intermedio — el hub está siempre desplegado.

Opción C
Pills + vert-col DERECHA (simétrico con Sort popup)
El hub es un vert-col position:absolute anclado a la DERECHA (el Sort popup lo tiene a la izquierda). Floats sobre el tab content — no lo desplaza. La fila del popup es: close-btn · pills · [vert-col flotante derecha].
